## Assignment instructions - Part 1: Design
# The first thing you should do before you start looking for data to use -- and before you start coding -- is to design your program. You should have done this several times before now, and it’s no different in your major assignment. 

Your design should include:

a statement of what your data is
the analysis you want to perform, and
how the data will be displayed.

Additionally, you should decompose the problem into functions, writing the function signature for each one you will need.

Remember: a function signature indicates the return type, function name, and any function parameters.

Some additional drawings can be made of your final visualisation. That will be the general design of how you will finally display your data. You should include graphical displays as well as possibly displaying some of your raw data.

You should submit any documents or images you generate with your code at the end of this assignment
