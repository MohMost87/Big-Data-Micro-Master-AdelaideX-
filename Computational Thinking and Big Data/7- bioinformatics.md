For every possible k-mer combination of length k (e.g., AAAAA, AAAAC, AAAAG...):
    Set count = 0
    Slide a window of length k across the DNA sequence from start to finish:
        If the current window matches the target k-mer:
            Increment count by 1
    Save the final count for this specific k-mer<br>
